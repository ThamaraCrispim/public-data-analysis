# public-data-analysis

Repositório dedicado a projetos de análise de dados utilizando exclusivamente Python e bases de dados públicas. Este portfólio tem como objetivo documentar análises exploratórias, gerar insights a partir de dados reais e demonstrar a aplicação prática de técnicas analíticas em diferentes contextos.

Os projetos aqui apresentados são desenvolvidos com foco em investigação orientada por dados, interpretação de padrões e construção de narrativas analíticas baseadas em evidências.

## Objetivo do repositório

Este portfólio foi criado para:

- Aplicar análise exploratória de dados em problemas reais
- Trabalhar com bases públicas de diferentes áreas
- Desenvolver raciocínio analítico e interpretação de dados
- Documentar processos de análise de forma clara e reproduzível
- Demonstrar evolução técnica em projetos de dados com Python

## Abordagem analítica

Cada projeto é conduzido seguindo princípios de análise de dados aplicados:

- entendimento do contexto e do problema
- exploração inicial do dataset
- limpeza e tratamento de dados
- análise exploratória e identificação de padrões
- visualização e interpretação
- geração de insights e conclusões

## Fontes de dados

Os projetos utilizam dados abertos provenientes de fontes oficiais e institucionais, como:

- portais de dados abertos do governo
- instituições públicas
- bases estatísticas nacionais
- conjuntos de dados disponibilizados para pesquisa e transparência pública

## Projetos desenvolvidos

As análises contemplam diferentes áreas e domínios, incluindo:

- segurança pública e mobilidade
- indicadores sociais
- dados administrativos governamentais
- estudos exploratórios com séries temporais
- investigações comparativas entre períodos



O repositório é atualizado continuamente com novos projetos, melhorias nas análises existentes e aprofundamento das investigações conforme o avanço dos estudos e da experiência prática em dados.
