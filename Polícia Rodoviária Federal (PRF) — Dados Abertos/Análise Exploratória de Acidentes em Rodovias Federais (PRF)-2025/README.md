# Análise Exploratória de Acidentes em Rodovias Federais (PRF) – 2025

Este repositório contém uma Análise Exploratória de Dados (EDA) sobre acidentes de trânsito ocorridos nas rodovias federais brasileiras, utilizando dados públicos da Polícia Rodoviária Federal (PRF) referentes ao ano de 2025.

O projeto tem como objetivo responder algumas perguntas específicas a partir dos dados, aplicando técnicas básicas de análise exploratória e visualização.



## Objetivo

- Realizar uma análise exploratória dos dados de acidentes
- Responder perguntas definidas a partir do conjunto de dados
- Praticar análise de dados utilizando Python e bibliotecas populares
- Desenvolver raciocínio analítico e interpretação de resultados



## Perguntas de Análise

As análises realizadas neste projeto buscam responder às seguintes perguntas:

- Qual a distribuição dos acidentes ao longo do tempo (dias da semana e horários)?
- Quais estados (UFs) apresentam maior número de acidentes?
- Como se comporta a gravidade dos acidentes de acordo com sua classificação?
